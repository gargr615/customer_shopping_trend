# 🛍️ Customer Shopping Trends Dashboard

Welcome to the **Customer Shopping Trends Dashboard**!

This project presents an interactive Customer Shopping Trends Dashboard built in Power BI to analyze shopping behavior, purchase patterns, customer ratings, and payment preferences.
The dashboard provides valuable insights into customer purchasing trends, helping businesses make data-driven decisions.

## 🚀 Features

📈 Interactive charts and KPI visuals

🗺️ Geographic analysis of customer locations

⭐ Average customer review ratings

💳 Payment method filtering and analysis

📦 Purchase amount analysis by product size

🚚 Shipping type comparison

📊 Drill-down and filtering capabilities

📂 Clean and structured dataset integration


## 🛠️Tech Stack
Power BI – Dashboard creation & visualization

Microsoft Excel / CSV – Data source

DAX – Measures & calculations

Power Query – Data cleaning & transformation
